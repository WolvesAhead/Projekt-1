//Struktur //- Es wird Button geben um auszuwaehlen ob man alleine oder zuzweit spielen moechte. //- Es wird eine Tabelle geben um den Charakter auszuwaehlen. //- Es wird eine Bildflaeche fuer jeden Charakter in dieser Tabelle geben. //- Es wird zwei Textfelder geben um seinen Namen einzutragen. //- Es wird einen Button geben um zwischen den Schwierigkeiten Einfach, Mittel und Schwer auszuwaehlen //- Es wird einen Button geben um die Steuerung der Spielfiguren einzusehen. //- Es wird 8-Bit Hintergrundmusik geben //- Es wird fuer jede Karte ein Hintergrundbild geben.
